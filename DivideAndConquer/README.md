## 分解
将问题分成3个子问题

int mid = (low+hight)/2

- [low, mid]  ，同类子问题
- [mid+1, high]， 同类子问题
- 肯定包含mid和mid+1的子数组。带条件的特定子问题。
## 解决

分别解决3个子问题
同类子问题，直接递归解决。
### 递归结束条件
子数组只剩一个元素。
### 带限制条件最大子数组问题解决
肯定包含mid和mid+1。

分别算左右，再相加。

low==high
